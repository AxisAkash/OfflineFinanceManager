# Changelog

## [1.0.0] - 2026-06-26

### Added
- Project scaffolding with feature-first architecture
- PIN and biometric authentication
- SQLite database with full schema and indexing
- Transaction CRUD with filtering, sorting, and aggregation
- Multiple wallet management with balance tracking
- Category-based budget planning with progress visualization
- Dashboard with balance overview, savings rate, and recent transactions
- Analytics with category spending breakdown
- Savings goals with progress tracking
- Debt management with payment tracking
- Recurring transactions support
- Backup and restore functionality
- Dark mode with system theme support
- Skeleton loaders for all async operations
- Empty state and error handling for all screens
- TypeScript strict mode throughout
- ESLint configuration
- Comprehensive documentation

### Security
- PIN hashing with SHA-256 and salt
- Secure Store for sensitive data
- Biometric authentication support
- App auto-lock on background
